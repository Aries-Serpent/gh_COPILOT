# SQL Recreation Prompt

Example from `documentation/HAT_COPILOT_USER_GUIDE.md`:

```text
GitHub Copilot Prompt:
"Create a SQL query that accomplishes the following:
- Query type: SELECT
- Purpose: User data retrieval with filtering
..."
```

## Usage
1. Comment the prompt in your SQL file or script.
2. Invoke GitHub Copilot to produce the query.
3. Verify the generated SQL before running it.
